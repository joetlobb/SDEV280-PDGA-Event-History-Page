# PDGA-Event-History-Page
PDGA Event History Page

The platform provides interactive data visualization and event history insights for the Professional Disc Golf Association.

Tech Stack: JavaScript, HTML, CSS, PHP, MySQL, ECharts.js

Key Contributions:
• Collaborated with Data Analysts and Software Developers from PDGA
• Engineered, developed, and analyzed the application in an Agile environment
• Integrated ECharts.js for advanced visualizations
